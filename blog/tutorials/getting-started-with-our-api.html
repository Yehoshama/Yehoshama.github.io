Getting Started with Our API

March 2026 · 8 min read

This guide walks you through setting up and making your first API call with the Yehoshama API.

Prerequisites

Before you begin, make sure you have:

  • An active Yehoshama account
  • An API key (generated from your dashboard)
  • curl or a REST client installed

Authentication

All API requests require authentication via an API key in the Authorization header: 

Authorization: Bearer YOUR_API_KEY

Making Your First Request

Use the following endpoint to verify your connection:

GET https://api.yehoshama.com/v1/status

A successful response will return:

{ "status": "ok", "version": "1.0.0" }

Next Steps

Once authenticated, explore the full API documentation to build your integration. For support, reach out via our contact page.
